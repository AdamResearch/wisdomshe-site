Internal Admin

管理后台入口

这是 shedatahk limited 的内部工作台入口。公开客户只需要看公司首页和对应业务线； 内部团队则在同一个后台里统一处理实业 IP 与投资组合管理两条线的申请、客户、订单、付款与待办。

进入后台登录 返回公司首页
如果你是公开访客，请返回首页并进入对应业务线；如果你是内部团队成员，请直接登录后台。
后台处理什么
  • 两条业务线的公开提交与客户分流
  • 订单、付款、客户状态与待办事项
  • 共享内容、方法库与内部推进节奏
为什么是一个后台
  • 外部客户看到不同业务线，内部仍由同一家公司统一承接。
  • 不同团队处理不同业务线，但不需要切换两套系统。
  • 线索、收款、交付与跟进能保持在同一条经营链路里。
